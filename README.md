## Como publicar no GitHub Pages

1. Crie um repositório público no GitHub, por exemplo: `portfolio-gustavo`.
2. Envie `index.html`, `estilo.css`, `script.js`, `README.md` e a pasta `assets` para a raiz do repositório.
3. Abra **Settings > Pages**.
4. Em **Build and deployment**, escolha **Deploy from a branch**.
5. Selecione a branch **main**, pasta **/(root)** e clique em **Save**.
6. Aguarde a publicação e abra o endereço informado pelo GitHub Pages.

O endereço normalmente ficará parecido com:

`https://SEU-USUARIO.github.io/portfolio-gustavo/`

## Observação sobre o formulário

O formulário possui validação visual em JavaScript. Como o projeto é estático e não utiliza servidor, ele não envia mensagens para um e-mail real.
